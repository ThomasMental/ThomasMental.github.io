## Directory Structure 文件夹结构

刚建立博客，我们目前的文件夹目录应该为:

```
  .
  ├─ _posts
  │  ├─ 2021-04-17-welcome-to-jekyll.markdown
  ├─ .gitignore.txt
  ├─ _config.yml
  ├─ 404.html
  ├─ about.markdown
  ├─ Gemfile
  ├─ Gemfile.lock
  ├─ index.markdown
```

本文介绍通用的Markdown语法，平台特有不涉及。

## Headers 标题

使用#代表标题级别，具体语法为: 

``` markdown
  # 一级标题
  ## 二级标题
  ### 三级标题
  #### 四级标题
  ##### 五级标题
```

效果如下:

### 三级标题

#### 四级标题

##### 五级标题


## Emphasis 强调

```
  **加粗**
  _斜体_
  _斜**加粗**体_
```

效果如下:

**加粗**
_斜体_
_斜**加粗**体_

## Strikethrough 删除线

```
  ~~装逼~~学习
```

效果如下:

~~装逼~~学习
